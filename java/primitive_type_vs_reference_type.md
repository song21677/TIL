## primitive type
* 변수에 값을 저장한다.
* 종류
  * 논리형 : boolean
  * 정수형 : byte, short, int, long
  * 실수형 : float, double
  * 문자형 : char

<br><br>
## reference type
* 변수에 객체의 주소를 저장한다.
* 기본값 : null
* 종류
  * 배열
  * 클래스
  * 인터페이스
